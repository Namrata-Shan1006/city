City-wide Traffic Signal Optimization System

‚≠ê Explanation

Optimize traffic flow by dynamically adjusting signal timings at intersections based on congestion levels.

üßÆ Algorithm Used

You can maintain a heap of intersections with max congestion to decide which signals to change first.

for(int i = n/2; i >= 1; i--) {
    int k = i;
    int v = H[k];

        int j = 2*k;
        if(j < n && H[j] < H[j+1]) j = j+1;

        else {
            H[k] = H[j];
            k = j;
        }
    }

}

Disaster Management & Evacuation Route Planner

‚≠ê Explanation

Find the safest and fastest routes for evacuation during emergencies, minimizing travel time and avoiding hazards.

üßÆ Algorithm Used

Dijkstra's Algorithm
Evacuation is a shortest-path problem on weighted road networks; Dijkstra efficiently finds minimal travel distance from source to all destinations.

üíª C++ Code

int dist[MAXN], path[MAXN], visited[MAXN];

for(int i = 0; i < n; i++) {
    dist[i] = cost[source][i];
    path[i] = source;
    visited[i] = 0;
}
visited[source] = 1;

for(int iter = 0; iter < n-1; iter++) {
    int u = -1, min = INF + 1;
    for(int i = 0; i < n; i++) {
        if(!visited[i] && dist[i] < min) { min = dist[i]; u = i; }
    }
    if(u == -1) break;
    visited[u] = 1;
    for(int v = 0; v < n; v++) {
        if(!visited[v] && cost[u][v] != INF) {
            if(dist[u] + cost[u][v] < dist[v]) {
                dist[v] = dist[u] + cost[u][v];
                path[v] = u;
            }
        }
    }
}

üßÆ Algorithm Used

The primary function is instantaneous look-up. Hashing allows mapping a unique item ID (the key) to its precise physical location (the value) in O(1) average time, which is essential for rapid retrieval and sorting.

üíª C++ Code

// key is the Item ID. hashFunction(key) gives the starting index.

    if (hashTable[index].key == key) {
        return hashTable[index].value; // Found the item location
    }
    index = (index + 1) % TABLE_SIZE;
    if (index == start_index) {

    }

City-Wide Electricity Grid Fault Detection System

‚≠ê Explanation

Monitors the entire power network in real-time to instantly detect line breaks or faults and identifies which substations or homes have lost connection due to the event.

üßÆ Algorithm Used

Union-Find
Union-Find (Disjoint Set Union) is perfectly suited for efficiently tracking the connectivity of the grid (a graph). It can quickly determine if two components (e.g., a power plant and a neighborhood) are still in the same connected set after a fault occurs.

üíª C++ Code

C++ 
// parent[] stores the parent of each node.
int find_set(i) {
    if(parent[i] == i)
        return i;
    return parent[i] = find_set(parent[i]);
}

void union_set(int i, int j) {
    int root_i = find_set(i);
    int root_j = find_set(j);
    if(root_i != root_j) {
        parent[root_i] = root_j;
    }
}

üßÆ Algorithm Used

Flood modeling requires determining all-pairs reachability (which point can reach which) or all-pairs shortest paths to see how blockages affect the entire flow network simultaneously, efficiently handled by Floyd-Warshall (O(V¬≥)).

üíª C++ Code

C++ 
void floyd_warshall(int n) {
    // k is the intermediate node
    for (int k = 0; k < n; k++) {
        // i is the source node
        for (int i = 0; i < n; i++) {
            // j is the destination node
            for (int j = 0; j < n; j++) {
                if (dist[i][k] != INF && dist[k][j] != INF) {
                    if (dist[i][k] + dist[k][j] < dist[i][j]) {
                        dist[i][j] = dist[i][k] + dist[k][j];
                    }
                }
            }
        }
    }
}

üíª C++ Code

int partition(float arr[], int low, int high) {
    float pivot = arr[high]; 
    int i = (low - 1); 

    for (int j = low; j <= high - 1; j++) {
        if (arr[j] > pivot) {
            i++;
            swap(&arr[i], &arr[j]);
        }
    }
    swap(&arr[i + 1], &arr[high]);
    return (i + 1);
}

Smart City Public Complaint Prioritization System

‚≠ê Explanation

This system automatically scores incoming public complaints based on urgency and impact. It then feeds them into a dynamic queue to ensure the most critical issues are routed to service teams first.

üßÆ Algorithm Used

Heapification (Max-Heap)
The system requires a Priority Queue where the highest score (most urgent complaint) is always extracted first. Heapification is the efficient process (O(n)) used to build or maintain the Max-Heap structure for this queue.

üíª C++ Code

C++ 
// n is the size of the heap, k is the index to sift down
void max_heapify(int n, int k) {
    int v = H[k];
    int j = 2 * k;

    while (j <= n) {
        // Find the larger child
        if (j < n && H[j] < H[j + 1]) {
            j = j + 1;
        }
        
        if (v >= H[j]) {
            break;
        } else {
            H[k] = H[j];
            k = j;
            j = 2 * k;
        }
    }
    H[k] = v;
}
