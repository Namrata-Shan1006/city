University Admission & Hostel Allocation System

‚≠ê Explanation

The system handles student application ranking based on merit scores, assigns admissions, and then efficiently allocates limited hostel rooms based on student priority (rank, distance, or preference) to maximize resource utilization.

üßÆ Algorithm Used

Quick Sort / Max-Heap
The system uses Quick Sort (O(N log N) average) for efficient ranking of applicants. The subsequent hostel allocation is managed by a Max-Heap (O(log N)) to consistently retrieve the highest-priority student for room assignment.

üíª C++ Code

C++ 
void quickSort(int H[], int low, int high) {
    if (low < high) {
        int pi = partition(H, low, high); 
        quickSort(H, low, pi - 1);
        quickSort(H, pi + 1, high);
    }
}

Smart Tourism Planner for Coastal Zone

‚≠ê Explanation

The system finds the shortest route sequence between a starting point and multiple Points of Interest (POIs), calculating the minimum travel time or distance to optimize the tourist's itinerary.

üßÆ Algorithm Used

Dijkstra's Algorithm
Dijkstra efficiently finds the Single-Source Shortest Path (O(E + V log V)) with non-negative weights.

üíª C++ Code

C++ 
const int V = 4;
const int INF = 99999;

void dijkstra(int src, int adj[V][V], int dist[]) {
    int sptSet[V] = {0};
    for (int i = 0; i < V; i++) dist[i] = INF;
    dist[src] = 0;

    for (int count = 0; count < V - 1; count++) {
        int u = minDistance(dist, sptSet);
        sptSet[u] = 1;

        for (int v = 0; v < V; v++) {
            if (!sptSet[v] && adj[u][v] != INF &&
                dist[u] != INF && dist[u] + adj[u][v] < dist[v]) {
                dist[v] = dist[u] + adj[u][v];
            }
        }
    }
}

Binary Search

üíª C++ Code

C++ 
int binarySearch(const int IDs[], int N, int targetID) {
    int low = 0;
    int high = N - 1;

    while (low <= high) {
        int mid = low + (high - low) / 2;

        if (IDs[mid] == targetID) return mid;
        else if (IDs[mid] < targetID) low = mid + 1;
        else high = mid - 1;
    }
    return -1;
}

‚≠ê Explanation

Manages citizen queues using priority handling to minimize wait times and improve service.

üßÆ Algorithm Used

Max-Heap
Highest priority citizen always served first.

üíª C++ Code

C++ 
const int N = 5;
int H[N + 1] = {0, 5, 3, 1, 9, 7};

void maxHeapify(int H[], int k, int n) {
    int v = H[k];
    int heap = 0;

    while (!heap && 2 * k <= n) {
        int j = 2 * k;
        if (j < n && H[j] < H[j + 1]) j = j + 1;

        if (v >= H[j]) heap = 1;
        else {
            H[k] = H[j];
            k = j;
        }
    }
    H[k] = v;
}

void serveNextCitizen(int H[], int *n) {
    if (*n < 1) {
        printf("Queue is empty.\n");
        return;
    }
    
    printf("Now Serving Priority: %d\n", H[1]);
    H[1] = H[*n];
    (*n)--;
    maxHeapify(H, 1, *n);
}

ATM & Kiosk Maintenance Business

‚≠ê Explanation

Minimizes total cable length or travel cost across ATM/Kiosk locations.

üßÆ Algorithm Used

Prim's Algorithm
For MST generation to minimize total connection cost.

üíª C++ Code

C++ 
const int V = 4;

int primMST(int adj[V][V]) {
    int key[V];
    int mstSet[V] = {0};
    int totalCost = 0;

    for (int i = 0; i < V; i++) key[i] = 99999;
    key[0] = 0;

    for (int count = 0; count < V; count++) {
        int u = minKey(key, mstSet);
        if (u == -1) break;

        mstSet[u] = 1;
        totalCost += key[u];

        for (int v = 0; v < V; v++) {
            if (adj[u][v] != 0 && !mstSet[v] && 
                adj[u][v] < key[v]) {
                key[v] = adj[u][v];
            }
        }
    }
    return totalCost;
}

const int NODES = 5;
int parent[NODES];

int find(int i) {
    if (parent[i] == i) return i;
    parent[i] = find(parent[i]);
    return parent[i];
}

void unite(int i, int j) {
    int root_i = find(i);
    int root_j = find(j);

    if (root_i != root_j) parent[root_i] = root_j;
}
